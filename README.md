# Task Manager - React + Bootstrap App

A fully functional task management application built with React and Bootstrap 5. Features add/edit/delete tasks with nested workloads, and responsive design.

## Features

- **CRUD Operations**: Create, read, update, delete tasks
- **Double-click Edit**: Double-click any row to edit task
- **Workloads**: Each task can have multiple workloads (start/end dates)
- **LocalStorage**: Data saves across browser sessions
- **Bootstrap 5**: Responsive design, modals, tables, forms

## Quick Setup

### 1. Clone Repository
```
git clone https://github.com/vuhvp/SPINTASK.git
```

### 2. Install Packages
```
npm install
```

### 3. Start the app
```
npm start
```
